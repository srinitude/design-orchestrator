# Code of Conduct

This project is intended to be a practical, respectful space for improving an open-source Agent Skill.

## Expected Behavior

- Be respectful and constructive.
- Keep feedback specific and actionable.
- Stay focused on the skill package and its users.
- Avoid sharing private, proprietary, or sensitive design material in public discussions.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic disruption.
- Misusing the project to expose private screenshots, credentials, customer material, or proprietary references.

## Enforcement

Maintainers may remove comments, close issues, block participants, or reject contributions that violate this code of conduct.

Report sensitive concerns using the guidance in [SECURITY.md](SECURITY.md).

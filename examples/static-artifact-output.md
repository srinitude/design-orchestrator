# Static Artifact Output Example

Route: `static-visual-artifact`

## Reference Notes

- Audience: technical attendees scanning a hallway poster between sessions.
- Use context: the poster must work from 10 feet away and still reward close reading.
- Visual signal: AI safety benefits from precision, restraint, and visible systems thinking.
- Accessibility: high contrast and generous type scale matter more than dense detail.

## Design Brief

Create a poster direction for an AI safety workshop. The artifact should communicate seriousness, technical depth, and open collaboration. The output is a design direction and generation prompt, not a rendered image.

## Viewer Path

1. First read: workshop title and date.
2. Second read: core theme and host credibility.
3. Close read: three session tracks and location details.
4. Exit cue: registration URL or QR placement.

## Visual Direction

Use a precise grid with one dominant title block, a narrow technical notation column, and a quiet diagrammatic field that suggests evaluation, containment, and review. Keep the palette restrained: off-white ground, graphite text, one safety accent, and one muted secondary tone. Avoid sci-fi glow, alarmist red, and generic neural-network decoration.

## Message Hierarchy

- Level 1: workshop title.
- Level 2: date, city, host.
- Level 3: theme statement.
- Level 4: three tracks.
- Level 5: registration details.

## Prompt Set

```text
Design a premium technical conference poster for an AI safety workshop. Use a precise editorial grid, large readable title typography, sparse technical notation, and a restrained diagrammatic field suggesting evaluation and containment. Palette: off-white, graphite, one calm safety accent, one muted secondary tone. Avoid sci-fi glow, generic neural networks, alarmist red, decorative gradients, and dense body copy.
```

## Critique

The concept is clear because it treats the poster as an information object, not a moodboard. The main risk is overloading the technical notation column; keep it secondary and never let it compete with the title.

## QA Checklist

- Legibility: title readable from distance, details readable close up.
- Hierarchy: five message levels are visually distinct.
- Composition: QR or URL has enough quiet space.
- Accessibility: contrast remains high in print and screen preview.
- Handoff: prompt includes concrete exclusions and composition constraints.
